abstract fun requestLocation(success: (Coordinate) -> Unit = {}, failure: (error: LocationProviderError) -> Unit = {})
Content copied to clipboard

Request device's location.

Request returns only one location and then stops.

Parameters

success

callback that provides location as Coordinate that contains Latitude and Longitude.

failure

callback that provides LocationProviderError.

Request will fail with LocationProviderError.AnotherRequestInProgress if another LocationProvider request is in progress. Request will fail with LocationProviderError.LocationServicesNotEnabled if location services not enabled at the device's settings. Request will fail with LocationProviderError.LocationPermissionsNotGranted if AuthorizationStatus is not AuthorizationStatus.AUTHORIZED. Request will fail with LocationProviderError.Unknown if unexpected exception occur during request.
