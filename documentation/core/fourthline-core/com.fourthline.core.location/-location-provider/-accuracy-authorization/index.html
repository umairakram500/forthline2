Returns the information about the calling application's access to an accurate location. This property should be interpreted only after app requested authorization from the user. If AuthorizationStatus is in state AuthorizationStatus.NOT_DETERMINED then accurate location information cannot be retrieved even if AccuracyAuthorization is FULL.
