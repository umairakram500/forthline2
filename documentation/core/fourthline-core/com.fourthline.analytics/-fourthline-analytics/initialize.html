Initializes Fourthline Analytics and starts collecting data.

  • Important: Even though analytics data starts being collected, it is not sent to Fourthline until TrackingConsent is set to granted. Use setTrackingConsent to update the tracking consent to TrackingConsent.GRANTED once the user has agreed to the terms and conditions of your application.

Parameters

tenantId

The analytics tenant id received from Fourthline.

context

Used to create local cache.
