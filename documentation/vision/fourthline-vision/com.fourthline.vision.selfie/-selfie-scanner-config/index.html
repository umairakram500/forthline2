Class which allows customisation of SelfieScanner behaviour.

Parameters

debugModeEnabled

Defines if scanner will display debug view. Default value is false.

recordingType

Defines if scanner will record and which type of recording it will perform. Default value is RecordingType.VIDEO_ONLY.

livenessCheckType

Determines which liveness check SelfieScanner does after SelfieScannerStep.SELFIE step. Default value is LivenessCheckType.HEAD_TURN.
