Class which represents step of DocumentScanner. Note: Any step timeouts in 40 seconds.

Parameters

index

Position in the total number of document scanner steps, 0-based.

fileSide

Determines file side of document page.

isAngled

Determines if document page should be tilted (some security features can be seen only when document is tilted).

isAutoDetectAvailable

Determines if DocumentScanner` will try to scan page automatically.

Note: We advise to hide snapshot button when this property is set to true and show it after some timeout (10-15 seconds), assuming that user cannot do automatic scan of document page with MRZ.
