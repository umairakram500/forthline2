List of supported countries.

Can be created by calling next method with json data object retrieved from GET /v1/lists/nfcInformation REST API: supportedCountries = CountryNetworkModel.create(jsonData)

Note: Providing this data will enable additional checks. DocumentScannerStepResult.validationErrors will contain DocumentValidationError.ISSUING_COUNTRY_NOT_SUPPORTED, DocumentValidationError.DOCUMENT_TYPE_NOT_SUPPORTED or DocumentValidationError.NATIONALITY_NOT_SUPPORTED when corresponding check fails for MRZ retrieved from document.
