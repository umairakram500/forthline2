Class which represents data retrieved during DocumentScanner scanning process.

Parameters

videoRecording

Container for the recorded video and metadata. It will be null if DocumentScannerConfig.recordingType is set to RecordingType.NONE.

mrzInfo

MRZ information retrieved from automatic scanning or null if there is not automatic scanning process or it was manually triggered. See MrzInfo

documentType

Document type passed in DocumentScannerConfig.

stepResults
