data class DocumentScannerConfig(type: DocumentType, includeAngledSteps: Boolean, debugModeEnabled: Boolean, recordingType: RecordingType, mrzValidationPolicy: MrzValidationPolicy, validationConfig: DocumentValidationConfig, videoDuration: VideoDuration)

Class which allows customization of DocumentScanner behaviour.

Parameters

type

DocumentScanner determines steps based on document type.

includeAngledSteps

DocumentScanner includes angled step after each non-angled step when this variable is set to true. Default value is true.

debugModeEnabled

Defines if scanner will display debug view. Default value is false.

recordingType

Defines if scanner will record and which type of recording it will perform. Default value is RecordingType.VIDEO_ONLY.

mrzValidationPolicy

DocumentScanner rejects any MRZ which does not compliant with policy, It will return best MRZ candidate retrieved in reasonable time. Default value is MrzValidationPolicy.STRONG.

validationConfig

DocumentScanner performs document step data validation based on this configuration object.

videoDuration

Represents the maximum duration of a video recording. Default value is VideoDuration.DEFAULT.
