Defines the colors used for the InputFields within Orca.

Parameters

textColor

Color used by the input field's text. Default is OrcaPalette.dark.

placeholderColor

Color used by the input field's placeholder. Default is dark 60%.

titleColor

Color used by the input field title on the top. Default is OrcaPalette.darkLight.

statusColor

Color used by the input field hint message below the field. Default is OrcaPalette.darkLighter.

errorColor

Color used by the input field's error message below the field.

backgroundColor

Color used for the input field background. Default is OrcaPalette.light.

borderColor

Color used by the input field unfocused border. Default is OrcaPalette.gray.
