Parameters

isLight

Boolean that defines if this Palette is intended for light- or dark-mode usage.

primary

Color used for illustrations and elements that require focus such as buttons / inputs. Examples: Animations, icons, primary button background, secondary button text, second option text primary button, scanner progress, table selectors, scanner checks, links.

accent

Color used for some UI elements to display choice.

dark

Color palette dark. Mainly used for default heading color.

darkLight

Color palette dark light. Mainly used for default text color.

darkLighter

Color palette dark lighter. Mainly used for default alternative and less emphasis text color.

darkSoft

Color palette dark soft.

light

Color palette light. Mainly used for default background color.

gray

Color palette gray.
