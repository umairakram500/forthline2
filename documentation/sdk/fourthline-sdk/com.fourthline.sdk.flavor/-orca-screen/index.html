Defines the colors used for the screens within Orca.

Parameters

backgroundColor

Default is OrcaPalette.light.

headerColor

Color used by the screen header. Default is OrcaPalette.dark.

titleColor

Color used by the screen title. Default is OrcaPalette.dark.

messageColor

Color used by the screen message. Default is OrcaPalette.darkLight.

cellBackgroundColor

Cells background color. Default is OrcaPalette.light.

tableElementTextColor

Color used by the table elements cells text. Default is OrcaPalette.dark.

tableElementIconColor

Color used by the table elements cells icon. Default is OrcaPalette.primary.
