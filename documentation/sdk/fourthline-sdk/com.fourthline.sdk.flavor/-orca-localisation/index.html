Defines the localisation scheme to be used by Orca.

Parameters

baseLanguage

The default language to be used in case the users' device language is not supported and is not defined in LanguageType.

fixedLanguage

Set a specific language to be used by Orca. The user interface will be presented using this localisation. baseLanguage and the users' device language will be ignored.
