images

required, list should NOT be null or empty and should contain properly arranged Attachment.Documents. There is a special validation rule for lists that contain angled images (see Attachment.Document.isAngled). Every such image should be preceded with a normal non-angled image of the same Attachment.Document.fileSide. For example:

Valid:

{[fileSide = FRONT, isAngled = false]};

{[fileSide = FRONT, isAngled = false], [fileSide = BACK, isAngled = false], ...};

{[fileSide = FRONT, isAngled = true], [fileSide = FRONT, isAngled = false], ...}.

Invalid:

{[fileSide = FRONT, isAngled = false], [fileSide = FRONT, isAngled = true], ...};

{[fileSide = FRONT, isAngled = true], [fileSide = BACK, isAngled = false], ...}.
